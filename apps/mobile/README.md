# Pravado Mobile

Expo React Native application for Pravado v2.

## Development

```bash
# Start development server
pnpm dev

# Run on iOS simulator
pnpm ios

# Run on Android emulator
pnpm android

# Run in web browser
pnpm web
```

## Build

```bash
# Type check
pnpm typecheck

# Build TypeScript
pnpm build
```

## Note

This is a stub implementation for Sprint S0. Full mobile features will be implemented in future sprints.
